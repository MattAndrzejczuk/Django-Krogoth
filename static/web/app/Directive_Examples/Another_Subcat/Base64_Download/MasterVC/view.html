Asynchronous Image Loading As Base64
This Example Will Load A Colossal 30+MB Image And Render It Smoothly

By pressing the button below, you will begin the download of a very large image file in Base64 format. With Base64, you'll be able to save the image to Local Storage giving Krogoth the capability to use extreme-quality media in the GUI by recycling it from the user's hard disk storage.ngh

Async Messages:

Button will trigger GET request to: /moho_extractor/LoadFileAsBase64?name=

Django view logic can be edited at /moho_extractor/views.py with the class named LoadFileAsBase64View

The interesting thing about this example code is that images or any other type of file could be loaded by using the WebSockets which are included to work out of the box for the Krogoth Framework.
