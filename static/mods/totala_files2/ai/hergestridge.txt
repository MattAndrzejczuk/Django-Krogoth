// This AI profile is designed to be used in maps with 
// almost no water mases (sea, huge lakes or similar 
// water areas). It is based on ground units but
// air units will be present as air support. This profile
// is a non-cheated AI and is designed to play with OTA
// units. Bug Fix patch goes very well. UH patch could 
// work fine too, but it's not tested. The rest of 3rd party units
// haven't been tested and supported. Only KrogTaar
// heavy kbot (by Silencer - TAP) is supported, but it's not
// a problem if you haven't installed the KrogTaar.

// Next, I give some numbers to show you the behaviour of this AI.
// These numbers are only a guide and could be imprecise.
// But remember that the developement of the AI depends
// on the resources amount and the map type.
// Metal maps offers the most similar numbers to the given ones.

// Within marks 00:07:00 & 00:10:00 - First attack. The closer bases, the earlier attack.
// Within marks 00:10:00 & 00:15:00 - Upgrade to #2 tech. level. Plasma & heavy laser defences.
// About mark 00:15:00 - Fusion plants (could be delayed if there's no enough resources).
// About mark 00:25:00 & higher - Masive #2 level tech. attacks.
// About mark 00:30:00 - AI starts to build Berthas/Timmys. First air attacks.
// About mark 00:40:00 - Berthas/Timmys fully operative.
// About mark 00:45:00 - Continuous air & ground sieging. Nuke silos are under construction.
// About mark 00:50:00 & 01:00:00 - Nuke silos fully operative. Krogoth/KrogTaar assembled.
// Within marks 01:01:00 & 01:30:00  - Massive nuke deployment. No survivors(?)...
