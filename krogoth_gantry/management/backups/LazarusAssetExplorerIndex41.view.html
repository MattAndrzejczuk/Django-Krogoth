Lazarus Terminology

Mod

Mods are like a table of contents that tell Lazarus which Assets are associated with whichever mod.

Asset

Assets are representations of whole Total Annihilation units. An Asset contains a unique identifier, which allows dependencies to find a TA unit it can associate with.

Dependency

Dependencies represent the pieces that make up Total Annihilation units. A dependency can be a unit model, weapon model, wav file sound effect, animation script and more. They contain information about where the raw 3DO file or WeaponTDF is stored on the ArmPrime server.

Total Annhilation will crash if a dependency is either missing, or if the dependency conflicts with another pre-existing dependency. Lazarus will do it's best to preserve the integrity of all dependencies.
