Project Status There are currently 3 major components of Lazarus that will need to be completed before the public beta test. 1. Mod Publisher 2. Download TDF Editor 3. Weapon TDF Editor

The main dashboard page of a custom Lazarus mod called "ArmNCorePack". This is the starting point for any new mod project that shows an overview of the currently selected mod project and a main menu to navigate to all primary Lazarus apps.

Mod Publisher Finalizes your mod project and compresses it into a .UFO file.

This is where you build and compile your .UFO file when you're ready to publish and play your custom TA mod. The the most important component of Lazarus overall.

This portion of the application modifies the weapon objects which are assigned to each unit in Total Annihilation. This editor will function in two modes, official '3.1' mode and unofficial '3.9.02' mode. Thanks to the incredible work done to the Total Annihilation unofficial patch, '3.9.02' mode will allow mod creators to have more than 255 unique weapon IDs. The only drawback to this mode (hopefully for the time being) is that it will only be playable on Microsoft Windows and not runable on macOS and Linux.

The major advantages of the new weapon TDF editor over anything that exists today will be a GAF file picker menu and a sound picker menu. The GAF picker will allow you to modify the explosion animations of weapons and have the ability to preview the GAF animations before you select them. The sound picker will work just like the GAF picker, but it will deal with the sounds that weapons make on impact and launch.

Initially you will only be able to pick official Cavedog sounds and GAF animations, but this will be updated in the future to where Arm Prime will have an entire GAF animation repository mod developers in the community will be able to select from.
