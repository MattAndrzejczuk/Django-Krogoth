TOTAL ANNIHILATION LAZARUS v{{ vm.lazarusVersion }} BETA        
Lazarus is almost ready for building and playing Total Annihilation mods. Posted by Matt on September 14, 2017

The project is nearly ready to be tested alongside Total Annihilation. Just today I managed to complete the weapon editor API controllers on the frontend. Although obviously, it will need an overhaul on the user experience side of things since currently it will not display relevent information to help users navigate the weapons which are installed on their mod project. I'll be focusing more right now on making sure the tech tree editor is fully functional before I jump to polishing up the weapon editor.

The unit FBI editor is in a good enough state for the public beta test. During the public data test of Lazarus, I'll be allowing users to stress test all Lazarus apps from uploading HPI data to building and publishing their mods to test on their copies of Total Annihilation. When the public beta test ends, I'll most likely be wiping the database for a fresh clean slate and added improvements to the apps based on user test feedback.
